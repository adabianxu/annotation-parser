# 断头路测试数据

本目录包含用于测试的断头路标注数据：

1. **断头路标注规范评测集.md** - 标注规范文档
2. **label_rule_VLA_TSR_1STEP_NO_REFINE_1.json** - 配置文件
3. **69ae7003721b587cd25ee46f.json** - 示例数据1（有标注）
4. **69ae700383f2493f72e8fde1.json** - 示例数据2（脏签数据）

## 使用方法

在网页中依次上传：
1. 标注规范文档（Markdown）
2. 配置文件（JSON）
3. 示例数据（可多选JSON文件）

然后点击"开始生成文档"按钮。
